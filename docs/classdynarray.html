template<class T >
bool dynarray< T >::resize_array ( int  newsize)
private

Resize the array by the specified size.

The function creates a new array of the specified size and copies the contents from the old array to the new array. The old array is deallocated. The function also updates the size of the dynamic array to the new size. The newsize needs to be larger than the currently allocated size of the array.

Parameters
[in]newsizethe size of the reallocated array
Returns
true if successful, false otherwise
